# ProbeAI UI/UX Design Concepts

This directory contains three distinct design directions for the ProbeAI interface, based on the requirements in `docs/prd.md`. Each concept targets a different user need and interaction style.

## 1. [Zen Mode (The Focused Researcher)](./design-concept-1-zen-mode.md)
*   **Keywords:** Minimalism, Readability, Simplicity.
*   **Best for:** Executives, casual users, or those who want immediate results without distraction.
*   **Key Feature:** Single-column layout, distraction-free reading experience.

## 2. [Mission Control (The Intelligence Dashboard)](./design-concept-2-mission-control.md)
*   **Keywords:** Data-Dense, Transparency, Control.
*   **Best for:** Researchers, Analysts, Developers.
*   **Key Feature:** Grid layout showing real-time agent logs, search queries, and source maps.

## 3. [Co-Pilot (The Collaborative Assistant)](./design-concept-3-co-pilot.md)
*   **Keywords:** Conversational, Iterative, Interactive.
*   **Best for:** Brainstorming, Planning, Collaborative drafting.
*   **Key Feature:** Split-screen with Chat on one side and a Live Document Canvas on the other.
